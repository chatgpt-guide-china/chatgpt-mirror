## 什么是 ChatGPT镜像网站？

[**ChatGPT镜像网站**](https://chat.lanjingai.org)（ChatGPT Mirror）是指将原始官网的内容复制到另一服务器上的网站服务，用户在国内无需翻墙就能使用~

与官网相比，**ChatGPT镜像网站**更适合国内用户：

- **无需翻墙**：直接在国内使用，无需代理工具。
- **中文对话**：具备更流畅的中文对话能力，适合学习、办公和日常使用。
- **功能全面**：支持 AI 绘画、AI 知识库、AI 思维导图、AI 文件、AI PPT 等实用功能，更符合国内用户需求。
